# App Store release kit

This folder contains a practical release kit to publish the iOS app.

## Execution order

1. `account-onboarding-checklist.md`
2. `release-preflight-checklist.md`
3. `appstore-connect-metadata-template.md`
4. `upload-and-review-checklist.md`

## Required public links for App Store Connect

Host these files on your public domain and use their URLs in App Store Connect:

- `public/privacy-policy.html`
- `public/support.html`

## Notes

- Replace all placeholder emails/domains before submission.
- Keep App Privacy answers aligned with actual app and backend behavior.
